### Install virtualenv and virtualenvwrapper

### Local setup 

* clone this repository
* cd into ~/phthisis_website/tb_website/tb_website
* Set up the virtualenv and install requirements:

```
mkvirtualenv tb
pip install -r requirements/local.txt
```
